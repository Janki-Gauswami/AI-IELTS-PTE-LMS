# AI-Powered IELTS & PTE Learning Management System

## Description

A web-based AI-powered Learning Management System for IELTS and PTE coaching institutes.

## Features

- Student Management
- Teacher Management
- Batch Management
- Learning Materials
- IELTS Practice
- PTE Practice
- Analytics Dashboard
- AI Band Prediction
- AI Study Planner

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

AI:
- Python
- Scikit-learn
- Whisper
